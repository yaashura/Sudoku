# 🧩 Sudoku Game

A desktop Sudoku game developed using Java and Swing.  
The game includes multiple difficulty levels, timer system, and interactive UI.

## 🚀 Features

- 🎮 3 difficulty levels (Easy, Medium, Hard)
- ⏱️ Countdown timer system
- ✅ Real-time solution checking
- ❌ Wrong input highlighting
- 🏁 Game over and success screens
- 🔄 Restart game option
- 🎨 Custom UI with background images




# 🧩 Sudoku Oyunu 

Bu proje, Java ve Swing kullanılarak geliştirilmiş masaüstü bir Sudoku oyunudur.  
Farklı zorluk seviyeleri, süre sistemi ve etkileşimli kullanıcı arayüzü içerir.

## 🚀 Özellikler

- 🎮 3 farklı zorluk seviyesi (Kolay, Orta, Zor)
- ⏱️ Geri sayım (timer) sistemi
- ✅ Gerçek zamanlı doğru/yanlış kontrolü
- ❌ Hatalı girişlerin vurgulanması
- 🏁 Başarılı ve başarısız bitiş ekranları
- 🔄 Yeni oyun başlatma özelliği
- 🎨 Özel tasarlanmış arayüz ve arka plan görselleri
